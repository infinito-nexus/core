Follow the instructions from AGENTS.md, then implement the given requirement file by following the
procedure in `docs/agents/action/requirements.md` (relative to the Infinito.Nexus repository root)
exactly. That document is the source of truth; this skill only routes you
there. Begin by clarifying every open requirement with the `active-listening` skill (escalating any root-cause or design question you are not ~99% sure of to the `dialectic` skill), then work under the `robot` skill.
